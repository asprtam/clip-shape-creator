CSS Clip-path shape maker creator. Allows for simple creation of new baseline CSS shape function path.

The clip-path property allows you to make complex shapes in CSS by clipping an element to a basic shape (circle, ellipse, polygon, or inset) or complex shape (using new baseline css shape() function).

This page focuses solely on new baseline css shape function. The shape function can also be used in the offset-path property.

Hints:

Hold Shift ⇧ while dragging to make points snap to grid.

Hold Command ⌘ to extract curve control point from underneath.

By default when you are dragging path point ( not curve control point ) and the line is not curved you will also drag curve control points to keep the line flat.
In technical terms: you will also drag any adjacent ( connecting to next or prev point ) curve control points that are directly underneath this point.
Hold Option ⌥ to drag only the desired path point and allow curving the straight line.

Work in progress Press Option ⌥ when clicking on line beetween two points to add point between theese two points.

Work in progress Press Command ⌘ + Z to undo.
